模拟浏览器设置

所有需要用到模拟浏览器的Feed或者Action会在其设置页面有一个IE浏览器图标，点击即可设置。

可以设置的内容包括：

  1. UserAgent ：设置浏览器UA，选择预设的UA或自行输入。
  2. 最大执行时间 ：设置一次爬取最长耗时，执行时间超过这个值将返回超时失败。
  3. 使用JavaScript ：加载页面时执行包含的JavaScript代码，如无需要可关闭降低资源消耗。
  4. 加载图片 ：加载网页中的图片，如无需要可关闭降低资源消耗。
  5. Http Header ：发起的Http请求将附带的HTTP头，可包含Cookie等信息。格式为冒号分隔的多行Key和Value，可直接从Chrome控制台拷贝。
  6. 代理 : 设置代理，支持Socks和HTTP代理。
